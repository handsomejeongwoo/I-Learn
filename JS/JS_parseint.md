# parseint

```
parseint란 문자열을 숫자로 리턴해주는 함수이다.

EX) const age = '10';
console.log(PasrseInt(age)); //age= '10'

RESULT) 10
ParseInt는 문자열을 숫자로 return을 해주기 때문에 문자열10이 아닌 숫자 10이 출력된다.

다른예시
const myNumber = '3';
console.log(2 + ParseInt(myNumber));

RESULT) 5  //myNumber변수에 있는 문자열 3을 숫자로 반환시켜 2를 더해 5가 리턴이 되었다.
```
