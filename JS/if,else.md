# 조건문

## 조건문이란 주어진 참/거짓 여부에 따라 프로그램의 흐름을 결정할수 있는 구문을 뜻한다.

## 조건이란 불리언 데이터를 반환하거나 불리언 데이터로 해석할 수 있는 표현식을 의미한다.

# if문

## 키워드 if를 이용해 만드는 구문 'if문'은 가장 일반적인 형태의 조건문이다.

    if문 기본형태
    if(조건){

        // 조건이 true일때 실행

    }

# 사용예

    let number = 4
    if(number ==4){

        console.log("이정우 ㅎㅇ") //true이므로 이정우 ㅎㅇ가 출력

    }

# if else 사용하기

    let number = 5

    if(number ==3){

        console.log("넘버는 3이양")

    }
    else{

        console.log("넘버는 3이 아니양")

    }
